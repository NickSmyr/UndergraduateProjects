Άσκηση 6 - Inspect Element

Αρχική ιστοσελίδα

Τα αρθρα ανοικουν στην κλασση article-item

  1. Βάζοντας 100 px padding στο αριστερό άρθρο
  2.  Αλλάζοντας το περιεχόμενο του αριστερου άρθρου
  3. Ορίζοντας το font size, line height και spacing στο μέγιστο επιτρεπτο για το αριστερό άρθρο
  4. Αλλαγή μεγέθους εικόνων με χρήση γενικού selector img και !important
  5. Αλλάζοντας όλα τα layout να είναι inline με χρήση του css κανόνα

* {

display: inline;

}

Φαίνεται πως τα scripts και τα styles απο το head έχουν μπεί στην ιστοσελίδα ως elements

Φαίνεται

Αν αντι για

* {

display: inline;

}

Χρησιμοποιήσουμε

body * {

display: inline;

}
