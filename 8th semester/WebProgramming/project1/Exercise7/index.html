Το Deep Learning ασχολείται με την δημιουργία αλγορίθμουν που μαθαίνουν απο δεδομένα για να παράξουν κάποια συμπεριφορά. Αυστηρά μιλώντας, αποτελεί υποπεδίο της μηχανικής μάθησης· Οι αλγόριθμοι μαθαίνουν απο τα δεδομένα για να μεγιστοποιήσουν κάποια πιθανότητα. Ωστόσο μεγάλη έμπνευση ήταν και η βιολογία. Βασικό συστατικό του Deep Learning είναι οι νευρώνες, που ενεργοποιούνται όταν δεχτούν κάποια συγκεκριμένη είσοδο.

Τα προβλήματα που χειρίζεται

Τα προβλήματα που χειρίζονται τα συστήματα Deep Learning είναι κυρίως προβλήματα που οι άνθρωποι μπορούν εύκολα να κάνουν αλλά δεν υπάρχουν ξεκάθαρα προγράμματα που μπορούν να φτιαχτούν για να τα λύσουν.

Το πιο σημαντικό παράδειγμα είναι η υπολογιστική όραση. Το πρόβλημα είναι να φτιαχτεί κάποιο πρόγραμμα που να εντοπίζει αυτόματα αντικείμενα σε μια εικόνα. Για παράδειγμα στην Εικόνα 1 το πρόγραμμα θα έπρεπε να εντοπίζει πως υπάρχει ένας σκύλος. Ωστόσο φανταστείτε πόσο δύσκολο είναι να φτιαχτεί ένα απλό πρόγραμμα για αυτό τον σκοπό. Μπορούμε να πούμε πως αν τα 4 κεντρικά pixel είναι άσπρα, τότε η φωτογραφία είναι ενός σκύλου; Θα μπορούσαμε να πούμε πιο απλά, αν έχει κάπου στην φωτογραφία μια μουσούδα κατα πάσα πιθανότητα είναι φωτογραφία σκύλου. Αλλα καταλήγουμε πάλι στην υπολογιστική όραση. Το να βρούμε μουσούδες είναι εξίσου δύσκολο.

Μια ακόμα διάσημη χρήση για το deep learning είναι η χρήση για την επεξεργασία γλώσσας. Όπως και με την υπολογιστική όραση, αυτό είναι κάτι εύκολο για εμάς τους ανθρώπους. Αν μας ρωτήσουν κάτι μπορούμε να πούμε την γνώμη μας, αν ξέρουμε δύο γλώσσες μπορούμε να μεταφράσουμε απο μία άλλη και άν κάποιος μας ζητάει να κάνουμε κάτι και τον συμπαθούμε, πιθανώς να του κάνουμε την χάρη. Τα Deep Learning συστήματα για την επεξεργασία γλώσσας επικεντρώνονται κυρίως στην αυτόματη μετάφραση γλωσσών, στην δημιουργία κειμένου και την αναγνώριση και επεξεργασία φωνής και μερικά διακεκριμένα και διάσημα συστήματα είναι το Amazon Alexa για αναγνώριση και επεξεργασία φωνής και το Google Translate για την αυτόματη μετάφραση.

Τα αποτελέσματα του Deep Learning

Για να μετρήσουμε πόσο καλά τα πάνε τα συστήματα Deep Learning και γενικότερα τα συστήματα Μηχανικής Μάθησης στα παραπάνω προβλήματα, οι κοινότητα έχει μαζεψει πολλά δεδομένα έλέγχου για να βλέπει πόσο καλά τα πάνε. Στην ουσία παίρνουμε τον αλγορίθμο και τον τρέχουμε με εισόδους εικόνες για παράδειγμα. Βλέπουμε τις προβλέψεις του, για παράδειγμα αν μια εικόνα έχει σκύλο, και τις συγκρίνουμε με τα πραγματικά περιεχόμενα των εικόνων.

Στο ImageNet dataset, ενα τεράστιο σύνολο εικονών με ενδέιξεις για το τί περιέχουν, δοκιμάζονται διάφορα συστήματα με τον παραπάνω τρόπο. Το καλύτερο σύστημα σήμερα 5 Απριλίου 2021 έχει πετύχει Accuracy περίπου 90%. Αυτό σημαίνει πως 90% των φορών προέβλεψε σωστά τα περιεχόμενα των εικονών.

Παρομοίως στην επεξεργασία φυσικής γλώσσας, υπάρχουν πολλά dataset για να δοκιμάζουμε μοντέλα Μηχανικής Μάθησης. Στο υποπρόβλημα της αυτόματης μετάφρασης υπάρχει συγεκριμένα ένα dataset που περιέχει πολλά κείμενα σε δύο γλώσσες, Αγγλικά και Γερμανικά. Εδώ οι μετρικές είναι πιο περίπλοκες απο ένα ποσοστό επιτυχίας αλλα αυτή την στιγμή οι αυτόματη μετάφραση με Deep Learning είναι το status quo.
